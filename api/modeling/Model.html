Model#

class Model : public Matrix, public AttributeManagers::Base#

Optimization model object.

This class represents an optimization model and is mainly composed of attributes (see Param), variables (see Var) and constraints (see Ctr). It is used to represent an optimization problem given in the following form:

\[\begin{split}\begin{align} \textrm{minimize } & c^Tx \\ \textrm{subject to } & Ax \ge b \\ & x \in \mathbb R \\ & x_j \in \mathbb N \textrm{ for some } j \end{align}\end{split}\]
