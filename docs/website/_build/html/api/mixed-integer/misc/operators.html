Operator Overloading¶

Building expressions in idol can be done in a very natural way using the C++ operator overloading. The following tables summarize the operators that can be used with the different expression types.

The tables show the return type of the operator when applied to the different expression types. The first column of the table shows the operator applied to the expression type in the first row. The first row of the table shows the expression type of the operator applied to the expression type in the first column. The return type of the operator is shown in the cell at the intersection of the row and column.
